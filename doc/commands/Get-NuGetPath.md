# Get-NuGetPath
Type: Function
Module: [Bca.Nuget](../ReadMe.md)

Gets the executable path of NuGet.exe.
## Description
Gets the executable path of NuGet.exe.
## Syntax
```ps
Get-NuGetPath [<CommonParameters>]
```
## Examples
### Example 1
```ps
Get-NuGetPath
```
This example will return the path of NuGet.exe.
## Parameters
## Inputs
**None**

## Outputs
**System.String**
Returns a string containing the path to NuGet.exe.
